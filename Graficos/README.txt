Para visualizar os gráficos das estatísticas:

1) Abra o arquivo 'estatisticas.html' no Chrome ou Firefox;
2) Por meio do formulário, selecione o arquivo 'estatisticas.txt' gerado pelo programa;

Obs: Arquivos 'estatisticas.txt' gerados em Windows provavelmente não serão corretamente lidos no Linux e vice-versa!
